# SSHはSecure Shellの略
暗号や認証を用いて、安全に遠隔のコンピュータを使うためのプロトコル。及びプロトコルを使って遠隔のコンピュータにログインするためのコマンド。

# SSH
- デフォルトは22番ポート
- サーバーに入って作業したりするときに使う
- プライベートネットワーク内のファイルを触るときに、SSHしたサーバー経由でファイルを閲覧したりもする
